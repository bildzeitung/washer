Getting Started
===============

.. currentmodule:: washer

Installation
------------

This library support Python 3.7.

::

    $ pip install washer

You can also install this library from source. Visit
`the project's GitHub page <https://github.com/bildzeitung/washer>`__, and
clone the repository.

::

    $ cd washer/washer-client
    $ pip install -e washer

To install the demo programs,

::

    $ cd washer/washer-client
    $ pip install -e demo
